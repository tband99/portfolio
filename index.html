Overview:
Self-directed project to develop practical skills in autonomous flight systems by modifying a hobby-grade FPV drone for autonomous operation using ArduPilot and ROS-based control software.

  • Currently developing autonomous flight capabilities on a hobby-grade FPV drone while learning ROS and ArduPilot integration.
  • Focused on establishing a functional link between ArduPilot firmware and ROS for mission scripting and basic waypoint navigation.
  • Building a Gazebo-based simulation environment to validate control logic before flight testing.
  • Next steps include PID tuning for stable flight and initial sensor integration (IMU and LiDAR) to support real-time obstacle avoidance.
  • Long-term goal: incorporate computer vision and mapping for more advanced autonomous behaviors.
