CubeSat ADCS Development (Capstone Project) - In Progress

Overview: Developing a modular Attitude Determination and Control System (ADCS) for Utah Tech’s CubeSat imaging mission, providing 3‑axis attitude control and estimation with a reusable architecture.

  • Implementing Nonlinear Dynamic Inversion (NDI) and Multiplicative Extended Kalman Filter (MEKF) algorithms in MATLAB
  • Modeling spacecraft dynamics and sensor response for closed-loop simulation
  • Integrating IMU, magnetometer, sun sensor, and star tracker data for attitude determination
  • Designing/tuning control laws for reaction wheel and magnetorquer actuation
  • Validating via Monte Carlo simulation and 3-axis attitude testing

Skills & Tools: MATLAB/Simulink, NDI, MEKF, sensor fusion, spacecraft dynamics, orbital mechanics.
