Mechatronic Miniature Golf Green

Associated with Utah Tech University | STEM Outreach Project

Project Overview

Designed and built a laser-controlled gate mechanism for a miniature golf green using an Arduino-based mechatronic system. The setup was showcased during a STEM outreach event and interacted with over 300 participants, demonstrating real-time sensor feedback and motion control.

Engineering Focus

  • Developed an Arduino-driven control system to actuate a laser-activated gate
  • Designed and fabricated structural components using SolidWorks and laser cutting
  • Integrated sensor input and actuator control for reliable game operation
  • Tested and tuned system performance for consistent gameplay interaction

Skills Applied

  • Arduino programming and control systems
  • SolidWorks modeling and design for fabrication
  • Laser cutting and rapid prototyping
  • Sensor-actuator integration
